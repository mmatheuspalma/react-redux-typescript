### Goomer Plaza

Goomer Plaza é uma plataforma que lista restaurantes e seus respectivos cardápios;

#### Requisitos
Gerenciadores de pacotes como Npm ou Yarn ( de preferência Yarn, pela velocidade e segurança )


#### Instalando dependências

##### Npm
``js
   npm install 
``

##### Yarn
``js
   yarn install 
``

#### Rodando o projeto

##### Npm
``js
   npm start
``

##### Yarn
``js
   yarn start
``
